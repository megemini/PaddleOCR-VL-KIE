# OCR Test

## 使用方式

```bash
python test.py -m <mode> [options]
```

### 参数说明

| 参数 | 说明 |
|------|------|
| `-m, --mode` | 模式选择：`pixels` 分析字符密度，`distance` 计算编辑距离 |
| `--pred` | 预测字符串列表，用于 `distance` 模式 |
| `-u, --usage` | 显示使用示例 |

### 示例

```bash
# 分析 t1-t7 图片的字符像素密度
python test.py -m pixels

# 计算预测结果与标注的编辑距离
python test.py -m distance --pred "由日田正口" "正月日口田田由" "月由田由口田正日正"

# 显示帮助
python test.py -u
```
